# Rao Vat App

A beginners swift project to create a rao vat  application in Swift 5.

## Features
- JSON, Restful API
- Login/ Register
- Post news
- Create news
- Upload Image

### Create the basic App with the back-end CRUD, upload image throughout UIImagePickerControler to back-end server.
